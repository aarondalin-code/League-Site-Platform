# Teams page plan

This document describes the first version of the Teams page.

## Purpose

The Teams page should list all teams in the league.

## Main content areas

### 1. Page title
Should show:
- Teams

### 2. Team list
Should show:
- team name
- team description if available
- any basic team details available in the CSV data

Main data source:
- Teams.csv

### 3. Team links
Should support:
- linking to players for each team later
- linking to schedules for each team later

Main data source:
- Teams.csv
- Players.csv
- Games.csv

## First version of the Teams page

The first version should focus on:

1. showing all teams
2. showing team names clearly
3. leaving room for future team detail pages
