# CSV connection plan

This repo currently stores the league CSV exports in `data/csv/`.

## Current CSV files

- SiteConfig.csv
- Teams.csv
- Players.csv
- Games.csv
- Practices.csv
- Drills.csv
- Trivia.csv
- Content.csv

## Step-by-step plan

1. Keep the CSV files organized in `data/csv/`
2. Add scaffold files for the platform
3. Add scripts or app code that read the CSV data
4. Build pages and features from the CSV content
