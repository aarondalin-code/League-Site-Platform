# Site page plan

This document maps the league site pages to the CSV data.

## Main pages

### Home page
Purpose:
- show league name
- show season info
- show welcome content

Main data source:
- SiteConfig.csv
- Content.csv

### Teams page
Purpose:
- list all teams
- show team names and basic team info

Main data source:
- Teams.csv

### Players page
Purpose:
- list players
- connect players to teams

Main data source:
- Players.csv
- Teams.csv

### Schedule page
Purpose:
- show games by date
- show opponents and matchups

Main data source:
- Games.csv
- Teams.csv

### Practices page
Purpose:
- show practice dates and team practices

Main data source:
- Practices.csv
- Teams.csv

### Drills page
Purpose:
- show drills and drill categories

Main data source:
- Drills.csv

### Trivia page
Purpose:
- show trivia questions and answers

Main data source:
- Trivia.csv

### Content page
Purpose:
- show articles, updates, and static content

Main data source:
- Content.csv

## First version of the site

The first usable version of the site should include:

1. Home page
2. Teams page
3. Players page
4. Schedule page
