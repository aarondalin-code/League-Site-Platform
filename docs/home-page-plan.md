# Home page plan

This document describes the first version of the league site home page.

## Purpose

The home page should give visitors a quick overview of the league.

## Main content areas

### 1. Site header
Should show:
- site name
- season name
- main navigation links

Main data source:
- SiteConfig.csv

### 2. Welcome section
Should show:
- short introduction
- welcome message
- featured content if available

Main data source:
- Content.csv
- SiteConfig.csv

### 3. Quick links
Should link to:
- Teams
- Players
- Schedule
- Practices

Main data source:
- static links for now

### 4. Featured updates
Should show:
- latest article or announcement
- key league update

Main data source:
- Content.csv

## First version of the home page

The first version should focus on:

1. site name
2. season info
3. welcome text
4. links to teams, players, and schedule
