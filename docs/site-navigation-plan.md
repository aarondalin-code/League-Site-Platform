# Site navigation plan

This document describes the main navigation for the league site.

## Main navigation links

The first version of the site navigation should include:

- Home
- Teams
- Players
- Schedule
- Practices
- Drills
- Trivia
- Content

## First version navigation

The first usable version should prioritize:

- Home
- Teams
- Players
- Schedule

Secondary links can be added after the first version is working.

## Future navigation ideas

Later versions can add:

- team detail pages
- player detail pages
- filters
- search
