# Practices page plan

This document describes the first version of the Practices page.

## Purpose

The Practices page should show practice sessions by team and date.

## Main content areas

### 1. Page title
Should show:
- Practices

### 2. Practice list
Should show:
- practice date
- team name
- practice details if available

Main data source:
- Practices.csv
- Teams.csv

### 3. Future improvements
Should support later:
- filtering by team
- grouping by week or month

## First version of the Practices page

The first version should focus on:

1. listing practices clearly
2. connecting practices to teams
3. organizing practices by date
