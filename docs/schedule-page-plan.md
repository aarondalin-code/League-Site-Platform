# Schedule page plan

This document describes the first version of the Schedule page.

## Purpose

The Schedule page should show games in a clear date-based order.

## Main content areas

### 1. Page title
Should show:
- Schedule

### 2. Game list
Should show:
- game date
- home team
- away team
- matchup details

Main data source:
- Games.csv
- Teams.csv

### 3. Future improvements
Should support later:
- filtering by team
- grouping by week or month
- showing game results if available

Main data source:
- Games.csv
- Teams.csv

## First version of the Schedule page

The first version should focus on:

1. listing games clearly
2. showing team matchups
3. organizing games by date
