# Content page plan

This document describes the first version of the Content page.

## Purpose

The Content page should show site articles, updates, and static content.

## Main content areas

### 1. Page title
Should show:
- Content
- News
- Updates
- or another simple label later

### 2. Content list
Should show:
- title
- slug if needed
- content type
- content body or summary if available

Main data source:
- Content.csv

### 3. Future improvements
Should support later:
- featured articles
- categories
- latest updates

## First version of the Content page

The first version should focus on:

1. listing content items
2. showing titles clearly
3. leaving room for detail pages later
