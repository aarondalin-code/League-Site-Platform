# Players page plan

This document describes the first version of the Players page.

## Purpose

The Players page should list league players and connect them to teams.

## Main content areas

### 1. Page title
Should show:
- Players

### 2. Player list
Should show:
- player name
- team name
- any simple player details available in the CSV data

Main data source:
- Players.csv
- Teams.csv

### 3. Team grouping
Should support:
- grouping players by team
- filtering by team later

Main data source:
- Players.csv
- Teams.csv

## First version of the Players page

The first version should focus on:

1. showing all players
2. connecting each player to a team
3. keeping the layout simple
