# Page build order

This document describes the recommended order for building the first version of the site.

## Recommended order

1. Home page
2. Teams page
3. Players page
4. Schedule page
5. Practices page
6. Drills page
7. Trivia page
8. Content page

## Why this order

This order starts with the most important and visible pages first, then adds the supporting pages after the core site experience is working.
