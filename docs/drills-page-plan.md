# Drills page plan

This document describes the first version of the Drills page.

## Purpose

The Drills page should show drills and drill categories.

## Main content areas

### 1. Page title
Should show:
- Drills

### 2. Drill list
Should show:
- drill title
- drill category
- drill details if available

Main data source:
- Drills.csv

### 3. Future improvements
Should support later:
- filtering by category
- grouping drills by type

## First version of the Drills page

The first version should focus on:

1. listing drills clearly
2. showing drill categories
3. keeping the page easy to scan
